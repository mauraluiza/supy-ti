# Supy TI - Sistema de Suporte Técnico

## Visão Geral
Aplicação web moderna para gerenciamento de suporte técnico, desenvolvida com React e TypeScript.

## Funcionalidades
- Dashboard com visão geral de clientes e tarefas.
- Cadastro de clientes (Winfood e Cplug) com dados criptografados.
- Gestão de Tarefas vinculadas a clientes.
- Anotações rápidas (Informações Gerais).
- Design responsivo (Mobile/Web).
- Tema Dark/Light.

## Tecnologias
- React 18
- TypeScript
- TailwindCSS
- Supabase (Backend)
- Lucide Icons
- Date-fns

## Como Executar
1. `npm install`
2. `npm run dev`

## Deploy
Deploy automático via Vercel.
